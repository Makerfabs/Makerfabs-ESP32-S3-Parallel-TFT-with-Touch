# WebAssembly port

# Step

## Install SDL

[follow this](https://wiki.libsdl.org/SDL2/Installation)

## Install Emscripten SDK
[follow this](https://emscripten.org/docs/getting_started/downloads.html)

## Build and Run
1. `mkdir build`
2. `cd build`
3. `emcmake cmake ..`
4. `emmake make`
5. open `index.html` in your browser.
