## How to use example

Before you begin, please ensure that esp-idf is installed on your computer. If not, click [Get Started with esp-idf](https://wiki.makerfabs.com/Get_Start_with_ESP-IDF.html) to complete the installation.

Launch the project in Visual Studio Code.

Set espressif Device Target **ESP32S3**

![image-20251117152822653](https://easyimage.linwanrong.com/i/2025/11/17/pacvxe-0.webp)

Use USB-C to connect the board and PC, select the corresponding port and Flash Device.

![image-20251120163625935](https://easyimage.linwanrong.com/i/2025/11/20/r27snw-0.webp)
